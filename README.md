# 🎨 toPixel — K-Means 智能像素画生成器

个人练手小作品，一个基于 K-Means 聚类算法的图片像素画生成工具。

## ✨ 功能

- **智能颜色量化**：使用 OpenCV K-Means 聚类算法，将图片颜色精简为指定数量，生成风格化像素画
- **可调参数**：自由调整 K 值（颜色数量）和像素化尺寸，多次尝试找到最佳效果
- **背景透明**：基于 GrabCut 算法一键抠图，去除背景
- **白色描边**：为像素画添加白色边框
- **灰度转换**：一键转为黑白像素画
- **导出 PNG**：保存为透明背景的 PNG 图片

## 🛠️ 使用方法

1. 在 PyCharm、VS Code 等可运行 Python 的编辑器中打开项目
2. 安装依赖：

```bash
pip install pillow opencv-python numpy
```

3. 运行程序：

```bash
python toPixel.py
```

4. 点击 **📁 选择图片** 上传你想像素化的图片
5. 调整参数：
   - **K 值（颜色数量）**：控制色彩丰富度，越小颜色越少、风格越强烈（建议 4~16）
   - **像素化尺寸**：控制像素块大小，越大像素感越强（建议 8~32）
6. 点击 **✨ 生成像素画** 查看效果
7. 根据需要选择抠图、描边、灰度等附加功能
8. 点击 **💾 保存 PNG** 导出成品

> 💡 **建议**：多调整 K 值和像素化尺寸进行对比，不同图片适合不同参数，反复尝试能达到最好效果。
